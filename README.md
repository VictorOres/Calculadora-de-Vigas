# Calculadora-de-Vigas
Calculadora de vigas
Bem vindo a Calculadora de Vigas v0.1

abaixo teremos algumas orientações para utilização dessa calculadora de vigas:

Passo 0:

antes de utilizarmos essa calculadora precisamos instalar duas bibliotecas python são elas Pycba e Matplotlib

para instalá-las é preciso abrir o cmd no windows e usar o comando pip install pycba e depois o comando pip install Matplotlib

Passo 1:
OBS:Para funcionamento do programa o arquivo Calculadora de Vigasv0.1.py e o funcoes.py devem estar na mesma pasta para operação.
